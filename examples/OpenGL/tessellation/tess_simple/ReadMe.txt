
Cg 3.0 Simple Tessellation Example

Draws two 2x2 patches in the x/y plane, sharing an edge.
Requires support for gp5tcp and gp5tep tessellation profiles.
Inner and edge tessellation can be adjusted with +-[]{} keys.
Animation can be toggled with the space key.
Filled/wireframe mode can be toggled with the f key.
Control points can be selected with p, and translated with the arrow keys.
